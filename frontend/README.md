# Staking Platform - Frontend

Фронтенд-приложение для платформы стейкинга криптовалют. Приложение позволяет пользователям просматривать доступные токены, создавать и отслеживать стейкинги.

## Требования

- Node.js 16.x или выше
- npm 7.x или выше

## Установка

1. Клонируйте репозиторий:

```bash
git clone <repository-url>
cd <repository-directory>/frontend
```

2. Установите зависимости:

```bash
npm install
```

## Запуск приложения в режиме разработки

```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для production

```bash
npm run build
```

Собранное приложение будет размещено в директории `build`.

## Структура проекта

```
frontend/
├── public/              # Статические файлы
├── src/                 # Исходный код
```

## Основные функции

1. **Аутентификация**: Пользователи могут войти в систему с помощью криптокошелька.
2. **Просмотр токенов**: Просмотр доступных токенов с возможностью фильтрации по сетям.
3. **Стейкинг**: Создание новых стейкингов и просмотр существующих.
4. **Профиль**: Информация о пользователе и его стейкингах.

## Взаимодействие с API

Приложение взаимодействует с бэкэндом через API. Основные эндпоинты:

- `/users/me` - Получение информации о текущем пользователе
- `/users/login` - Аутентификация пользователя
- `/users/logout` - Выход из системы
- `/tokens` - Получение списка токенов
- `/staking` - Управление стейкингами

## Настройка прокси

По умолчанию приложение настроено на взаимодействие с API по адресу `http://localhost:8000`. 
Для изменения адреса API отредактируйте поле `proxy` в файле `package.json`.
